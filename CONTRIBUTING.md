## Option 2: Submit via pull request

If you prefer GitHub, you can contribute directly to the repository.

### Steps
1. Fork this repository
2. Add your portfolio to the appropriate category in the list
3. Make sure the link is live and publicly accessible
4. Submit a pull request with a short description of the portfolio

---

## Guidelines

Please ensure that:
- The portfolio belongs to a designer or creative professional
- The website showcases real work and thoughtful design
- The link is publicly accessible
- No low-quality templates, link farms, or spam
